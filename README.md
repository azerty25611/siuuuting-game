# Siuuuting!

敵の攻撃を避けながら弾を撃って敵を倒す2Dシューティングゲーム

---

## 操作キー

- **Enterキー**: ゲーム開始 / 再スタート  
- **上下左右の矢印キー**: 移動  
- **スペースバー**: 弾を発射  
- **Pキー**: 一時停止 / 再開  
- **Alt + Enter**: 全画面表示切替
